# Material de apoio — Aula 1 (Fundamentos de Testes Mobile)

Leituras obrigatórias pré-Aula 2. **Todas as fontes são públicas/open access.**

## Pirâmide de testes — teoria

- `fowler-2012-test-pyramid-bliki.pdf` — Fowler, M. (2012) *Test Pyramid* (bliki).
  - Fonte: https://martinfowler.com/bliki/TestPyramid.html
- `fowler-2018-practical-test-pyramid.pdf` — Fowler, M. (2018) *The Practical Test Pyramid*.
  - Fonte: https://martinfowler.com/articles/practical-test-pyramid.html

## Pirâmide mobile — Knott (substitui livro Pearson 2015)

Mesmo autor do livro *Hands-On Mobile App Testing*, em artigos públicos:

- `knott-2014-mobile-test-pyramid-MoT.pdf` — Knott, D. (2014) *The Mobile Test Pyramid*. Ministry of Testing.
- `knott-2018-3-steps-mobile-testing-strategy.pdf` — Knott, D. (2018) *3 Steps to Creating a Mobile Testing Strategy*. Applause.

## Acadêmico — testes mobile em escala

- `linares-vasquez-2017-continuous-evolutionary-large-scale-mobile-testing-ICSME.pdf` — Linares-Vásquez, M. et al. (2017) *Continuous, Evolutionary and Large-Scale: A New Perspective for Automated Mobile App Testing*. ICSME 2017.
  - Preprint do autor (W&M): https://www.cs.wm.edu/~denys/pubs/
  - Paper peer-reviewed — cumpre critério "≥1 fonte acadêmica" da Atividade 1.

## Google Testing — estratégia de testes em escala

- `google-testing-2010-test-sizes.pdf` — *Test Sizes* (Small/Medium/Large).
  - Fonte: https://testing.googleblog.com/2010/12/test-sizes.html
- `google-testing-2015-just-say-no-e2e.pdf` — *Just Say No to More End-to-End Tests*.
  - Fonte: https://testing.googleblog.com/2015/04/just-say-no-to-more-end-to-end-tests.html

## ISTQB — terminologia oficial (substitui livro Black/Van Veenendaal)

- `istqb-ctfl-syllabus-v4-overview.pdf` — ISTQB Foundation Level CTFL v4.0 (página oficial).
  - Fonte: https://www.istqb.org/certifications/certified-tester-foundation-level/
  - Syllabus completo disponível gratuitamente via ISTQB; PDF aqui é overview oficial.

## Por que ler?

| Material | Por quê |
|---|---|
| Fowler (2x) | Base conceitual: pirâmide clássica + versão moderna |
| Knott (2x) | Adaptação mobile da pirâmide + estratégia prática |
| Linares-Vásquez | Pesquisa peer-reviewed sobre automação mobile em escala (fundamenta Atividade 1) |
| Google Testing | Anti-padrão E2E excessivo + classificação de tamanhos de teste |
| ISTQB | Vocabulário oficial — todo o mercado QA usa |

> Quem não ler **fica perdido** na Aula 2 (setup + unit testing).
